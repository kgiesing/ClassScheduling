int main ( int  argc,
char *  argv[] 
)

Main method for the test suite.

Accepts any of the following command-line arguments, in any order. Passing no arguments results in tests being run for all components.

  • -c: run tests on a specific Class. If you use this argument, the next argument passed must be the name of the class to test. Using this argument will override any other arguments except -t.
  • -i: run tests on the Input component (Reader classes)
  • -n: run tests on the geNetic schedule generator
  • -o: (lower-case "oh"): run tests on the Output component (Writer)
  • -p: run tests on the Preprocessor component
  • -r: run tests on the gReedy schedule generator
  • -s: run tests on the Schedule object
  • -t: run sTress tests. These tests will be run on any of the components that are tested. If you do not pass this argument, stress tests will not be run at all; and if you do pass this argument, no tests other than stress tests will be run. (Note: the colon, ":", is not part of the argument.)
